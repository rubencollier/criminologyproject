The Criminology Project

By Ruben Collier

A project exploring crime data, blogs, and statistics dashboard.

Coded in Visual Studio Code, built with HTML and vanilla CSS, deployed with Cloudflare. All text is set in the Inter typeface.

Features

  • Dashboard

    A comprehensive criminology statistics dashboard with interactive charts.

  • Blog

    Insightful articles and blog posts.

  • About

    An in-depth explanation about of how the site was created.
